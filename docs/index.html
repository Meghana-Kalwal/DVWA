About DVWA

🎯

What is DVWA?

Damn Vulnerable Web Application (DVWA) is an intentionally vulnerable PHP/MySQL web application designed for security professionals to practice their skills legally in a controlled environment.

🐛

Known Vulnerabilities

  • SQL Injection
  • XSS (Reflected, Stored, DOM)
  • Command Injection
  • File Inclusion (LFI/RFI)
  • File Upload Bypass
  • CSRF
  • Brute Force
  • Insecure CAPTCHA
  • Hardcoded Credentials
  • Security Misconfigurations
